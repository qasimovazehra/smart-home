# smart-home
An Arduino-based automation system for hands-free access and weather protection. Uses ultrasonic sensors for automatic main entrance and garage doors via servo motors, and an integrated rain sensor to retract outdoor laundry from water damage.
Hardware Components

Here are the specific hardware components used to build this smart home prototype:
Microcontroller:
Arduino Uno :The core control unit that processes sensor data and sends commands to the servo motors.
Sensors:
HC-SR04 Ultrasonic Distance Sensors (2 pcs): Used for proximity detection at the garage door and the main home entrance.
Rain Detection Sensor (1 pc): Monitors weather status and sends digital signals to protect the clothes rack.
Actuators:
SG90 Micro Servo Motors (3 pcs):Precise mini motors used to physically operate the garage door, main entrance, and the clothes rack mechanism.
Prototyping & Connections:
MB-102 Solderless Breadboard (1 pc):Used to prototype and connect all components safely without soldering.
Ribbon Jumper Wires (Male-to-Male & Male-to-Female): Multi-colored cables used for routing logic pins, 5V power, and GND lines between the Arduino and components.
